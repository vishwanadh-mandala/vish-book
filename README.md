# VishBook 🌐

A lightweight, interactive social network built with React.

## Features
- 📰 News Feed with posts, likes, and comments
- 👥 Friends discovery page
- 👤 Personal profile page
- ✍️ Create, delete posts in real-time
- 💬 Comment on any post
- 👍 Like / unlike posts

## Getting Started

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Build for Production

```bash
npm run build
```

## Deploy

Deployed via [Vercel](https://vercel.com) — just connect the GitHub repo and it auto-deploys on every push.
